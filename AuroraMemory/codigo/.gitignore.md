# `.gitignore`

Arquivo de configuração `.gitignore` que define quais arquivos e diretórios o Git deve ignorar no projeto.

**Responsabilidade:** Evitar que artefatos de build, dependências, arquivos de ambiente e dados locais sejam versionados.

**Itens ignorados principais:**
- `node_modules/`, `.next/`, `build/`, `out/` — dependências e saída de build
- `.env*` — variáveis de ambiente (com opt-in manual)
- `.vercel` — configuração de deploy
- `*.tsbuildinfo`, `next-env.d.ts` — artefatos do TypeScript
- `src/data/platform-store.json` — dados mockados do banco local

**Relações:** Protege arquivos gerados por Next.js, TypeScript, npm/yarn e Vercel, além de dados sensíveis ou temporários da aplicação.
