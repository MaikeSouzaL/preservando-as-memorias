# AGENTS.md — Regras do Agente Next.js

**Responsabilidade:** Este arquivo define regras críticas para agentes de IA que trabalham com este projeto Next.js, alertando sobre diferenças em relação ao Next.js padrão.

**Conteúdo principal:**
- Aviso de que esta versão do Next.js possui **breaking changes** em APIs, convenções e estrutura de arquivos
- Instrução obrigatória: ler o guia relevante em `node_modules/next/dist/docs/` antes de escrever qualquer código
- Alerta para prestar atenção em avisos de depreciação

**Como se liga aos outros:** Este arquivo é uma diretiva para agentes/desenvolvedores, não um módulo de código. Ele afeta **todos os outros arquivos** do projeto, pois estabelece que o comportamento do Next.js pode ser diferente do esperado, exigindo consulta à documentação local antes de modificar qualquer componente, API ou página.
