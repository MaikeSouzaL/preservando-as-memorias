## Responsabilidade Principal
Arquivo de bloqueio de dependências gerado automaticamente pelo npm. **Não deve ser editado manualmente.** Garante instalações reproduzíveis do projeto Next.js "preservando-memorias".

## Dependências Chave (Produção)
- **Next.js 16.2.5** + **React 19.2.4** — Framework e biblioteca core
- **@supabase/supabase-js** + **@supabase/ssr** — Autenticação e banco de dados
- **Stripe** — Processamento de pagamentos
- **Resend** — Envio de e-mails transacionais
- **Upstash Redis + Rate Limit** — Cache e limitação de requisições
- **Howler** — Reprodução de áudio
- **QRCode** — Geração de QR codes
- **date-fns** + **react-datepicker** — Manipulação de datas

## Dependências de Desenvolvimento
- **Tailwind CSS v4** — Estilização utilitária
- **TypeScript 5** — Tipagem estática
- **ESLint 9** + **eslint-config-next** — Linting
- **pg** — Driver PostgreSQL (para consultas diretas)
- **Babel React Compiler** — Otimização de componentes

## Integração com o Projeto
- Versões fixadas garantem consistência entre ambientes
- Lockfile v3 com `requires: true` assegura resolução determinística
- Todas as dependências de produção e dev estão registradas com hashes de integridade
