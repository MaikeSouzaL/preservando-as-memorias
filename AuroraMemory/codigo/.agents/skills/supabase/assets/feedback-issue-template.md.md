# `.agents/skills/supabase/assets/feedback-issue-template.md`

Este arquivo é um **template de feedback** para reportar erros ou omissões na documentação de skills do Supabase.

**Responsabilidade principal:** Estruturar relatos de bugs de forma padronizada, capturando o contexto da tarefa, o erro da skill, o resultado esperado e a fonte documental.

**Seções chave:**
- **What happened:** Descreve o problema (tarefa, o que a skill disse, o que era esperado).
- **Source:** Identifica o arquivo e seção da documentação onde o erro foi encontrado.
- **Fix suggestion:** Campo opcional para proposta de correção.

**Como se liga aos outros:** Este template é consumido por skills (como `supabase`) para gerar issues de correção no GitHub, vinculando feedback diretamente a arquivos de referência (ex: `references/security-model.md`).
