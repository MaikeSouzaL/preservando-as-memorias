# `.mcp.json`

Arquivo de configuração do **MCP (Model Context Protocol)** para integração com o **Supabase**.

- **Responsabilidade**: Define o servidor MCP `supabase`, que permite que ferramentas de IA (como Claude Desktop) executem operações no Supabase via CLI.
- **Servidor**: `supabase` — usa `npx` para executar o pacote `@supabase/mcp`.
- **Parâmetros**: `command` (`npx`), `args` (`-y`, `@supabase/mcp`), `env` (`SUPABASE_ACCESS_TOKEN` — token de acesso vazio, a ser preenchido).
- **Conexão**: Este arquivo é consumido por ferramentas que suportam MCP (ex: Claude Desktop, Cursor) para habilitar comandos como `supabase:query`, `supabase:migration`, etc.
